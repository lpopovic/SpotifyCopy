## Build and Runtime Requirements
+ Xcode 9.3 or later
+ Swift 5 or later
+ iOS 14.0 or later
+ [CocoaPods](http://cocoapods.org/) 

## How to run 
1. Clone this repo
1. Open shell window and navigate to project folder
1. Run `pod install`
1. Open `Spotify.xcworkspace`
1. Run the project on selected device or simulator
